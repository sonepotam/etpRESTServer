Описание процесса подключения к тестовому стенду обмена данными согласно 44-ФЗ


Уважаемые коллеги!

Для подключения вам потребуются тестовые клиентский и серверный сертификаты. Для
проверки корректности подписи наших сообщений вам потребуется наш сертификат открытого ключа.
Их можно получить у наших технических специалистов, координаты которых указаны в конце письма.

Тестовая среда доступна по адресу https://etptest.abr.ru/callback
Для доступа к тестовой среде используется проверка клиентского сертификата. Для получения доступа необходимо прислать свой клиентский сертификат.
Используется версия протокола TLS 1.2.

Для диагностики и отладки подготовлены следующие ресурсы:

1.	Отладка бизнес-логики доступна по адресу:
http://etptest.abr.ru/callback - интерфейс шлюзового решения. Подключения осуществляется без TLS-соединения.

2.	Диагностика TLS-соединения доступна по адресам:
https://etptest.abr.ru:8443 – заглушка для проверки SSL/TLS. Используется только TLSv1.2. Возвращает страницу с текстом «TLS12».
https://etptest.abr.ru:9443 – заглушка для проверки SSL/TLS. Используются все версии SSL/TLS. Возвращает страницу с текстом «SSL».

3.	Доступность среды для тестирования бизнес-логики проверяется по следующим ресурсам:
http://etptest.abr.ru/ping - тестовая страница для проверки статуса приложения. Если среда доступна, то возвращает страницу с текстом «hello».
https://etptest.abr.ru/ping - тестовая страница для проверки статуса приложения. Используется только TLSv1.2. Если среда доступна, то возвращает страницу с текстом «hello».


Для проверки подписи сообщений пришлите, пожалуйста ваш открытый ключ.
